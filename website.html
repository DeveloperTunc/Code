Front End

Front-End or Client-Side is where user interacts with provided webpage.When a user asks for some data, Front-End sends a request to Back-End, Back-End processes the data usually from its database and responses the relevant data to Front-End.

  • HTML
    • Provides website the skeleton architecture
    • Images, headers, links,SEO optimization , titles,included files, forms are some of its important aspects
  • CSS
    • Provides website the much needed design.Styling colors, images, links , everything you see on the web page designed and positioned by CSS.
  • JS
    • JavaScript adds the functionality to the webpage.We can design buttons with CSS and HTML but the button needs to actually do something which is provided by JS.JS can also make DOM manipulations to manipulate how webpage interacts.

Back End

Back-End or Server-Side is where magic happens when it comes to manipulating data.If i wanted to do a TO-DO app i would need to handle get requests to show all the queries in a list.If i wanted to update that list Back-End is where i would need to write the logic for that.Server-Side is where database models are defined.Back-End handles request that comes from Front-End processes data gets data from database if necessary and sends a response back to Front-End.

  • Python
    • Python can handle requests and responses , interact with database, define business logic.
